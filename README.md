# Pre-entrega.Wade
Este proyecto consiste en un inicio para un viejo proyecto de diseño UX UI que se llamó **CellArt**, es una plataforma dedicada a la venta y gestión de comisiones de arte digital, retratos personalizados e ilustraciones. LA plataforma es un intermediario para asegurar la venta segura y confiable de los servicios de los artistas y de sus clientes.
